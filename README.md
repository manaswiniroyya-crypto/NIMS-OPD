# Easy Navigation Within NIMS Hospital

## Overview

Easy Navigation Within NIMS Hospital is a web-based wayfinding system designed to help patients, visitors, and staff navigate the Nizam's Institute of Medical Sciences (NIMS), Hyderabad.

The project provides an interactive and user-friendly platform that enables users to locate departments, OPD blocks, emergency facilities, parking areas, pharmacies, restrooms, and other important locations within the hospital campus.

Unlike traditional navigation systems, this project uses real-world GPS coordinates and photographs collected through on-site visits to ensure accuracy and reliability.

---

## Problem Statement

NIMS Hospital consists of multiple interconnected blocks including:

- OPD Block
- Specialty Block
- Millennium Block
- Emergency Block
- Trauma Care Center

Due to the large campus layout, first-time visitors often face difficulties locating departments and facilities, leading to:

- Increased stress
- Delayed appointments
- Missed consultations
- Frequent dependency on hospital staff for directions

This project addresses these challenges through a digital navigation solution.

## Usage

1. Open the website.
2. Search for a department.
3. Select the desired location.
4. View:
   - Department details
   - OPD timings
   - GPS coordinates
   - Navigation instructions
5. Follow the step-by-step directions to reach the destination.
